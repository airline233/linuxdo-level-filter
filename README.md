# linux.do 等级贴过滤

一个给 [linux.do](https://linux.do) 用的油猴脚本，让你在帖子列表里一键按用户等级（Lv1 / Lv2 / Lv3）筛帖子。

## 为啥要有它

L 站左上角其实自带按分区筛选的入口，但它只能一个子版块一个子版块地点——福利羊毛 Lv1、福利羊毛 Lv2、跳蚤市场……你想「把所有 Lv2 的帖子拢到一起看」，就得挨个翻，没有一个统一的等级开关。

这脚本干的就是这件事：不管帖子来自哪个子版块，只看它的等级门槛。想只看 Lv1 就勾 Lv1，想把公开水帖藏掉就取消勾选「公开」，列表实时更新。

## 装它

1. 先装个用户脚本管理器，[Tampermonkey](https://www.tampermonkey.net/) 或 [Violentmonkey](https://violentmonkey.github.io/) 都行。
2. 点这里安装脚本 → [linuxdo-level-filter.user.js](./linuxdo-level-filter.user.js)（管理器会自动认出来）。
3. 刷新 linux.do，左上角会冒出来一个小工具条。

## 怎么用

**按等级筛**：工具条上四个 chip——`公开` `Lv1` `Lv2` `Lv3`。

- 点亮的会显示，熄灭的会从列表里藏起来。
- 默认全亮，也就是啥都不过滤，跟没装一样——你按需关掉不想看的就行。

**按标题关键词筛**：后面跟着 `黑` `白` 两个按钮和一个输入框。

- 输入框填关键词，多个词用逗号或空格隔开，不区分大小写，边打边生效。
- `黑`（黑名单）：标题命中任一关键词的帖子藏掉。
- `白`（白名单）：只留标题命中关键词的帖子，其余藏掉。
- 输入框空着时这个功能不生效，只走等级过滤。
- 等级和关键词是「都得满足」的关系——先过等级这关，再看关键词。

选择和输入都会记住，下次打开还是上回的设置。工具条左边有个 `⠿` 把手，按住能拖到任何顺手的位置，位置也会记住。

在首页、最新、分类页这些帖子列表页都生效。凡是没有帖子列表的页面（帖子详情、设置、搜索等）工具条会自动隐藏；个人资料页虽然也有列表，但同样不显示，不碍事。

## 它是怎么判断等级的

L 站的受限分区徽章上其实把等级写得明明白白，脚本按三条线索去读，取到一个就算数：

1. 徽章的 `title`，比如「此处为 1级用户 可见空间。」
2. 徽章文字尾巴上的 `Lv1`
3. 分区链接 slug 里的 `-lv1`

读不到锁的就当作公开帖。整个过程只读页面上已经渲染好的内容，不额外发任何请求。

## 已知的边角

- 只认得 Lv1 / Lv2 / Lv3 这三档。要是 L 站以后加了别的门槛档位，得再调。
- 万一哪个受限分区的徽章写法比较特别、没被正确归档，欢迎开 issue 把那段徽章的 HTML 贴过来，补一条正则就好。
